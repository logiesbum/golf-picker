# ⛳ Caddie — Golf One-and-Done Analytics Dashboard

A fully interactive golf pick'em analytics dashboard. Track picks, view AI-powered rankings, and analyze SG stats for every PGA Tour and LIV player.

## Setup (2 minutes)

### Step 1 — Upload to GitHub
- Create a new repository on GitHub (name it anything, e.g. `golf-dashboard`)
- Upload the `index.html` file to the repo

### Step 2 — Enable GitHub Pages
- Go to your repo → **Settings** → **Pages**
- Under "Source" select **Deploy from branch**
- Set branch to **main**, folder to **/ (root)**
- Click Save — your site will be live in ~60 seconds at `https://yourusername.github.io/golf-dashboard`

### Step 3 — Add your Anthropic API Key
- Open your live dashboard URL
- Click **⚙️ API Settings** in the bottom-left sidebar
- Paste your key (get one free at console.anthropic.com → API Keys)
- Key is stored only in your browser — never sent anywhere except Anthropic

## That's it. Bookmark the URL and it works on any device.
